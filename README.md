#  Location Tracker

A web-based Location Tracker project built using HTML, CSS, JavaScript, and Python. This application provides a user interface to track and display location-related information.

##  Features

-  Home page with navigation
-  About and Contact sections
-  Privacy policy page
-  Location tracking page
-  Basic interaction using JavaScript
-  Backend logic handled via Python script (`ttrack.py`)
-  Custom styling for each page
-  Image assets for enhanced UI

##  Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Python
- Images and media
